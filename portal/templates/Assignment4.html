3. (20 points) A Turing machine with doubly innite tape is similar to an ordinary Turing machine, but
its tape is innite to the left as well as to the right. The tape is initially lled with blanks except for the
portion that contains the input. Computation is dened as usual except that the head never encounters
an end to the tape as it moves leftward. Show that this type of Turing machine recognizes the class of
Turing-recognizable languages.
Solution: A Turing Machine with doubly innite tape can simulate an ordinary Turing Machine. The
rst tape of the 2-tape Turing Machine is written with the input string and the second tape is left blank.
Then the doubly innite tape Turing machine is cut into two parts, at the starting position of the input
string. The portion with the input string and all the blanks to its right appears on the rst tape of the
2-tape Turing Machine. The other part to the left of the input string appears on the 2nd tape in reverse
order.
